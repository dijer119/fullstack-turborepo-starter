ROE 컬럼 + 수동 입력 — Design Spec

Date: 2026-05-26

Scope: /stocks 페이지에 ROE 컬럼 추가. 자동 추정(PBR/PER) 표시 + 사용자가 셀에서 직접 수정 가능. 서준식 적정가/지수 계산이 수동값 우선 사용.

Goals

  • ROE를 표 컬럼으로 노출 — 서준식 계산식의 핵심 변수를 사용자가 직관적으로 확인.
  • 사용자가 수동 ROE를 입력해 서준식 계산을 재현 — 자동 추정의 한계(PBR/PER 기반) 보완.
  • 자동 갱신 배치가 수동 입력값을 덮어쓰지 않게 보호.

Non-Goals

  • 일괄 입력 / CSV 임포트 — YAGNI.
  • 수동값 변경 이력 추적 — 최신 값만 저장.
  • 외부 API에서 정확한 ROE 가져오기 — 별도 작업.

Data Model

새 테이블 StockOverride 추가. StockMaster와 1:1 옵셔널 관계.

model StockOverride {
  code      String   @id
  manualRoe Float?   @map("manual_roe")
  updatedAt DateTime @updatedAt @map("updated_at")

  master    StockMaster @relation(fields: [code], references: [code], onDelete: Cascade)

  @@map("stock_overrides")
}

단위: % 단위로 저장 (예: 9.98 = 9.98%). UI 표시와 일관. 계산식 적용 시 /100.

Server Actions

새 파일 src/actions/stock-overrides.ts:

  • setManualRoe(code: string, roe: number | null): Promise<void>
    • roe === null → row 삭제 (자동 추정으로 복귀).
    • roe !== null → upsert. 음수도 허용 (적자 ROE 케이스).
    • !/^\d{6}$/.test(code) → no-op (방어).

API 확장

StocksExplorerRow에 manualRoe: number | null 필드 추가. getStocksExplorer의 include에 override: true 추가.

UI

컬럼 위치

PBR 옆 (PER · PBR · ROE · 배당% · 안전마진 …). Valuation 비율 그룹과 자연스럽게 일치.

표시 규칙

상태표시스타일
수동값 있음12.50%파란색 + 굵게 + title="수동 입력"
자동 추정만9.98%회색 + 일반
PER/PBR 없음—흐린 회색

Inline Edit 동작

  1. 셀 클릭 → <input type="number" step="0.01"> 표시. 자동값을 placeholder, 수동값이 있으면 value로.
  2. Enter 또는 blur → setManualRoe(code, value) 호출. Optimistic update.
  3. 빈 값 + Enter → setManualRoe(code, null) (수동값 삭제, 자동 추정 복귀).
  4. Esc → 취소 (저장 안 함).
  5. 저장 중 input disabled. 실패 시 console.error + 이전 상태 복원.

서준식 계산 연동

seojunsikValue / seojunsikIndex 함수가 effectiveRoe = manualRoe ?? autoRoe를 사용하도록 변경. 호출부에서 r.manualRoe를 추가로 전달.

구현 형태 (% → 소수 변환 주의):

function effectiveRoe(manualRoe, per, pbr) {
  if (manualRoe != null) return manualRoe / 100;       // 9.98 → 0.0998
  if (per != null && pbr != null && per > 0 && pbr > 0) return pbr / per;
  return null;
}

검증 기준

  • 마이그레이션 성공, stock_overrides 테이블 존재.
  • tsc 통과 (EXIT=0).
  • /stocks 페이지가 ROE 컬럼 표시 — 자동값 정상 렌더, PER/PBR 없는 행은 —.
  • 임의 종목 셀 클릭 → input 표시 → 12.5 입력 후 Enter → 파란색 굵게 표시 + 새로고침 후에도 유지.
  • 같은 셀 클릭 → 빈 값 입력 후 Enter → 자동값(회색)으로 복귀.
  • 서준식 적정가/지수가 수동 ROE 반영 (입력 전후 값이 변함).

Out of Scope (메모)

  • 외부 ROE 데이터 소스 통합 — DART/네이버 정확 ROE는 별도 작업.
  • 수동 입력 ROE에 대한 검증 한계치 — 사용자 자유 입력. -100 ~ +200 같은 sanity check은 추후 필요시.
