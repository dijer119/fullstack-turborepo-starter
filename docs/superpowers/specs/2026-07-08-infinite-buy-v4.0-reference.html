무한매수법 V4.0 일반모드 규칙 정리 (학습 노트)

출처: 라오어 카페 "무한매수법 V4.0 방법론 _ (일반모드)". 본 문서는 학습·확인용이며 구현 전 원문 재검증 권장. 3배 레버리지 고위험 전략(투자 권유 아님).

0. 버전 위치 / 모드 개념

  • V4.0 = 일반모드 + 소진모드. 이 문서는 일반모드만 다룸(소진모드=리버스 무한매수법은 별도 글, 미확보).
  • 일반모드: 소진되기 전, 쿼터매도를 포함한 일련의 과정.
  • 소진모드: T값이 1회분 미만 남은 시점부터 시작 → 특정 조건 만족 시 일반모드로 복귀. (내용 미확보)
  • 오피셜 종목: TQQQ, SOXL만. 초심자는 TQQQ 권장, 익숙해지면 SOXL 겸행. (수익은 대개 SOXL이 높으나 변동성이 커 중도포기 위험)

요약 흐름도 (일별 사이클)

일반모드의 하루 루프 — 매일 같은 5단계. 공식·세부는 각 절 참조(T §2 · 별지점 §3 · 매수금 §4 · 매수 §5 · 매도 §6).

① T 갱신 — 전날 체결 반영 (§2)
② 계산 — 오늘의 별지점(§3) · 1회매수금(§4)
③ 주문 걸기 — 매수는 T 국면별(아래 표) · 매도는 ¼ 쿼터 LOC@별지점 + ¾ 지정가(§6)
장 마감 — 종가 vs 별지점, 한쪽만 체결
④a 종가 < 별지점
매수 체결(하락 담기) → T ↑
④b 종가 ≥ 별지점
쿼터매도 체결 → T ×0.75
⑤ 보유 0 → 사이클 종료
T=0으로 재시작(§7)
보유 > 0 → 내일 ①

별도 경로: 장중 지정가매도선(TQQQ 평단+15% / SOXL 평단+20%) 도달 시 ¾이 체결(T ×0.25). ④에서 매수·매도가 같은 날 겹치지 않는 이유는 매수점 = 별지점−0.01 (§3 예시 ②).

③의 매수 형태 — T 국면별 (전부 LOC, 아래 사다리 동반)

국면T 범위매수 주문
첫매수T = 0종가+10~15% 큰수
전반전0 < T < 분할/2½ 별지점 + ½ 평단
후반전분할/2 ≤ T ≤ 분할−1전액 별지점 (평단 아래)
소진T > 분할−1소진모드(리버스) 전환 — 별도 글

별지점 기준: 종가 위치별 판정

전반전(T<분할/2)엔 별지점 > 평단, 후반전(T≥분할/2)엔 별% 음수라 별지점 < 평단(평단보다 손절 시도). 3/4 지정가매도(TQQQ+15%/SOXL+20%)는 이 축과 별개로 장중 도달 시 체결.

1. 원금 · 분할 설정 (예전과 동일)

  • 원금 확정 후, 매수 잔금이 남아도 다른 투자에 쓰지 않음.
  • 분할: 20 / 30 / 40분할 권장. 분할↓ = 공격적·위험, 분할↑ = 방어적·안정.
  • TQQQ·SOXL 동시 진행 시 원금 독립 설정(한 계좌 OK). 같은 종목 다른 분할이면 계좌 분리 권장.

예: 2만달러 20분할 → 1회매수금 시작 $1,000 / 40분할 → $500.

2. T값 정의 (이전 버전과 다름 — 핵심 변화)

T = Turn(진행회차). 1.0T = 1회분 진행. 소수점 무제한 연속값(예: 5.165479525).

사건T 변화
1회치 매수 완료+1
절반치 매수+0.5
쿼터매도 발생직전T × 0.75 (곱셈)
지정가매도 후 LOC 매수(동시)(직전T × 0.25) + 1 또는 (직전T × 0.25) + 0.5

예: T=7 → 1회 매수 시 T=8 / 절반 매수 시 T=7.5 / 쿼터매도 시 T=7×0.75=5.25.

구조적 함의: T는 정수 회차가 아니라 실제 체결에 따라 변하는 연속값. 쿼터매도(3/4 잔존)면 ×0.75, 지정가매도(1/4 잔존)면 ×0.25. 매일 실제 매수·매도 체결량을 알아야 T를 갱신할 수 있음.

3. 별지점 (V2.2부터 도입, V4.0 핵심 기준)

매수·매도의 기준선. 별지점 = 평단 × (1 + 별%), 매일 계산.

별% 공식 (종목 × 분할별)

종목·분할별%
20분할 TQQQ(15 − 1.5·T) %
40분할 TQQQ(15 − 0.75·T) %
20분할 SOXL(20 − 2·T) %
40분할 SOXL(20 − 1·T) %

일반화: 별% = base × (1 − 2T/분할), base = TQQQ 15 / SOXL 20. → T = 분할/2(후반전 시작)에서 별% = 0%, 이후 마이너스(평단 아래).

  • 매수점 = 별지점 − 0.01 (매수·매도 겹침 방지), 매도점 = 별지점.
  • 후반전(T ≥ 분할/2)엔 별지점이 평단 아래 → "평단보다 손절을 시도"하는 성격.

예: 20분할 SOXL, 평단 $38.30, T=8.6 → 별% = (20−2×8.6)=2.8% → 별지점 = 38.30×1.028 ≈ $39.37. 이 값이 매수/매도를 가름.

별이란 무엇인가 — 직관

"별"은 별지점(★)을 가리키는 말로, 그날 하루의 매수·매도를 가르는 기준선 가격이다. 종가가 별 아래면 담고(LOC 매수 체결), 별 이상이면 보유의 1/4을 던진다(쿼터매도 체결). 핵심 성질은 두 가지:

  • 별은 평단에 붙어 다닌다 — 별지점 = 평단×(1+별%)이므로, 물타기로 평단이 내려오면 별도 함께 내려온다.
  • 별은 T가 커질수록 내려온다 — 별% = base×(1−2T/분할)는 T에 대한 감소 1차 함수. 사이클 초반엔 평단 위 높은 곳(익절 요구치가 높음), 반환점(T=분할/2)에서 평단과 일치, 후반엔 평단 아래(부분 손절도 감수)로 내려간다.
별의 역할: 오늘 시장이 어느 쪽으로 마감하든 자동으로 "옳은 행동"(하락 → 매집, 상승 → 부분 익절)이 나가도록 미리 걸어두는 스위치 가격. 참고로 첫 매수 직후(T=1, 40분할 TQQQ)의 별%는 14.25%로, 지정가매도선(평단+15%) 바로 아래에서 출발해 매일 조금씩 내려온다.

예시 ① 별의 일생 — T에 따른 하강 (40분할 TQQQ, 평단 $50 고정 가정)

T별% = 15 − 0.75T별지점 = 50×(1+별%)국면 · 의미

실전에선 평단도 매일 변하므로, 별지점은 평단의 이동 × T의 변화 두 힘의 합성으로 움직인다 (아래 예시 ③).

예시 ② 왜 매수점은 별지점 − 0.01인가

매수 LOC 리밋(매수점)과 매도 LOC 리밋(별지점)을 1센트 벌려 두면, 어떤 종가든 정확히 한쪽만 발동한다. 별지점 $53.75(매수점 $53.74)일 때:

종가쿼터매도 LOC @53.75매수 LOC @53.74결과
$53.75 이상체결 (종가 ≥ 리밋)미체결 (종가 > 리밋)매도만
$53.74 이하미체결 (종가 < 리밋)체결 (종가 ≤ 리밋)매수만

미국 주식 호가는 1센트 단위라 두 리밋 사이에 낄 종가가 없다 → 같은 날 매수·매도가 같은 가격에서 겹치는 일이 구조적으로 차단됨.

예시 ③ 하루 체결이 별을 움직이는 방향 (40분할 TQQQ)

오늘 상태: 평단 $50.00 · T=10 · 보유 100주. 별% = 7.5% → 별지점 $53.75(매수점 $53.74), 지정가매도선 $57.50(평단+15%). 1회매수금 ≈ $500 가정 → 주문: 매수 ½($250) LOC @53.74 = 4주 + ½ LOC @50.00 = 5주 (+아래 사다리), 매도 쿼터 25주 LOC @53.75 + 75주 지정가 @57.50.

오늘 종가체결내일 T내일 별지점해설
A. $49.00 (하락)매수 LOC 9주 전부 종가 체결 (사다리 미체결 가정)10+1 = 11평단 $49.92 × 1.0675 ≈ $53.29별이 내려온다 — 평단↓과 T↑ 둘 다 하강 방향
B. $54.20 (별 위 마감)쿼터매도 25주 @54.20 체결 (실현익 +$105), 매수는 전부 미체결10×0.75 = 7.5평단 $50.00 × 1.09375 ≈ $54.69별이 올라간다 — 매도는 평단을 안 바꾸고 T만 줄이므로, 다음 쿼터매도는 더 높은 종가를 요구
C. 장중 $57.50 터치지정가매도 75주 체결. 종가도 별 위면 쿼터까지 체결 → 보유 0, 사이클 종료종가가 매수점 아래로 급락 마감하면 LOC 매수 체결 → 계속: T = 10×0.25+1 = 3.5§11-4의 "지정가매도 후 LOC매수 동시" 케이스
비대칭성 요점: 매수 체결은 별을 끌어내리고, 매도 체결은 별을 밀어올린다. 별은 "방금 익절했으면 다음 익절 눈높이를 올리고, 물렸으면 눈높이를 낮추는" 자기조정 기준선이다.

예시 ④ 후반전의 별 — 평단 아래 부분 손절

평단 $50.00 · T=28 (40분할 TQQQ): 별% = 15 − 0.75×28 = −6% → 별지점 $47.00. 후반전 규칙대로 매수는 전액 LOC @46.99, 쿼터매도는 LOC @47.00 — 평단보다 $3 아래에 매도를 건다.

  • 종가 $47.50 마감 → 쿼터매도 체결(주당 −$2.50 손실 실현) → T = 28×0.75 = 21 → 별% = −0.75% → 별지점 $49.63. 손실을 잘라 T를 되돌리면 별이 평단 근처로 복귀한다.
  • 즉 후반전 쿼터매도는 단순 손절이 아니라 회차(T)를 되사서 소진(T>39)을 늦추는 디리스킹 비용으로 이해하는 게 정확하다.

4. 매수금 (이전 버전과 다름 — 매일 변동)

1회매수금 = 잔금 ÷ (분할 − T). (20분할: 잔금/(20−T), 40분할: 잔금/(40−T))

예: 원금 $20,000·40분할. 1회차 시도 $500 중 실제 $478 사용 → 잔금 $19,522, T=1 → 2회차 시도액 = 19,522 / (40−1) = $500.56. T가 +1/+0.5/×0.75 무엇이든 매일 재계산되어 미세 변동.

5. 매수 방법 (매일 걸어둠, 전부 LOC)

(1) 첫 매수 (보유 0, T=0)

  • 무조건 1회 매수 시도 → 전날보다 적당히 큰 값(종가 +10~15%)으로 LOC 매수(큰수) + 아래로 LOC 사다리.

예(1회 $617.89, TQQQ 종가 $45.93): 큰수 $51.44(≈+12%)에 12개 LOC + $47.53에 1개 + $44.13에 1개 … 종가 $44 마감이면 14개 체결.

(2) 전반전 매수 (T = 1 ~ 분할/2 미만)

  • 40분할이면 T=0~19.999…가 전반전.
  • 1회매수금의 절반 = 별지점 LOC, 나머지 절반 = 평단 LOC + 큰 하락 대비 아래 LOC 사다리 추가. 전부 LOC.

예(별지점 78.12, 평단 69.75): LOC 78.11에 3개 + 69.75에 4개 + 67.40에 1개 + 59.91에 1개 …

(3) 후반전 매수 (T = 분할/2 ~ 분할−1)

  • 40분할이면 T=20~39가 후반전. T > 39 초과 시 소진모드로 진입(별도 글).
  • 별지점이 평단 아래. 1회매수금 전체 = 별지점 LOC + 아래 LOC 사다리. 전부 LOC.

예(별지점 59.55): LOC 59.54에 9개 + 56.85에 1개 + 51.68에 1개 + 47.37에 1개 …

LOC 사다리란 무엇인가 — 직관 (2026-07-12 추가)

사다리 = 그날의 메인 매수(별지점/평단 LOC) 아래에 더 낮은 가격대로 여러 단 걸어두는 소량 LOC 매수. 매일 새로 걸고, 미체결 단은 그날로 소멸한다. 급락일수록 아래 단까지 연쇄 발동해 "더 싸게, 더 많이" 담는 장치다.

  • 체결 원리: LOC 매수는 종가 ≤ 리밋이면 리밋가가 아니라 종가로 체결된다. 계단식으로 걸어두면 — 보합·소폭 하락 마감엔 맨 위 단(별지점·평단)만, 급락 마감엔 종가보다 리밋이 높은 단이 전부 발동하고 모두 그날 종가(가장 싼 가격)에 체결된다.
  • 귀결: 하락 깊이에 비례해 매수량이 자동으로 늘어난다. 규칙의 개념적 기준은 별지점이지만 실제 주문장에 나가는 것은 매일 사다리 포함 다건 주문 — V2.2 대비 "급락장 매집력"(§9 장단점 표)이 바로 이 축이다. 세 국면(첫매수·전반·후반) 모두 "메인 + 사다리"가 한 세트.

예시 — 첫 매수 사다리의 체결 (1회분 $617.89, TQQQ 종가 $45.93)

주문리밋가수량다음날 종가 $44 마감 시
큰수 LOC (종가+12%)$51.4412개체결 @ $44
사다리 1단$47.531개체결 @ $44
사다리 2단$44.131개체결 @ $44

14개 × $44 = $616 ≤ 1회분 $617.89. 기본 수량(12개)을 리밋가($51.44) 기준으로 계산해 두면 실제 체결은 그보다 싼 종가로 이뤄져 예산이 남고, 사다리 단들이 그 잉여를 흡수한다. 단, 이는 원문 예시의 숫자가 맞아떨어지는 역산 해석이며 단 간격·단 수·수량 배분의 공식 규칙은 미확보(§11-3).

구현 주의: ① 배분 공식 미확보(§11-3) — 원문 예시 모사 수준에서 시작해야 함. ② 아래 단일수록 현재가 대비 괴리가 커져 매매거부(±20%) 한도와 충돌 가능(§8 큰수매수와 같은 계열 문제). ③ 하루 다건 동시 주문 + 부분체결 처리 필요(§10).

6. 매도 방법 (매일 걸어둠, 전·후반 공통)

  • 쿼터매도: 보유수량의 1/4을 별지점에 LOC 매도.
  • 최종매도: 나머지(3/4)를 지정가매도 — TQQQ 평단+15% / SOXL 평단+20%.
  • 매수와 달리 별지점은 LOC 매도, 최종은 지정가매도.
  • 지정가매도는 프리장(서머 17시/비서머 18시)에 걸어 프리~본장~애프터 전체 활용. 주간장 미사용. LOC는 예약/본장 아무 때나.

예(별지점 59.55): LOC 매도 59.55에 35개 + 지정가매도 75.76에 106개.

7. 종료 시점

  • 보유수량 0이면 종료.
  • 지정가매도 후 급락해 LOC 매수가 발생하면 종료 아님 → 계속 진행.
  • 종료 후 처음 매수로 복귀. 수익금 재투입(복리) vs 원금 유지(단리)는 선택. (저자는 1년 복리 후 원금 재설정)

8. 큰수매수 (방법론 무관, 매매거부 회피용)

증권사는 현재가 대비 약 20% 벗어난 주문을 매매실수 방지로 거부함. 3배 레버리지 하락장에서 이를 극복하려 큰수 매수 개념으로 주문을 성립시킴. (세부는 별도 카페글 — 미확보)

10. 구현 시 아키텍처 함의 (도입 결정 시 참고)

V4.0은 코어 함수 교체 수준이 아니라 영속적 fill-aware 상태가 필요:
  • T 영속화 + 체결 기반 갱신: 매수 체결량(+1/+0.5)·매도 종류(쿼터 ×0.75/지정가 ×0.25)를 매일 반영. 현재 코드는 매수 체결을 추적하지 않고 round만 증가시킴 → 신규 필요.
  • 잔금(cash) 영속화: 1회매수금 = 잔금/(분할−T). 사이클별 잔금 회계(원금−투입+회수) 필요. 현재는 원금 고정 + 계좌 매수가능금액 확인만.
  • 종목·분할을 전략 공식 입력으로: base(TQQQ 15/SOXL 20), 분할값이 별%·매수금에 직접 사용.
  • LOC 사다리: 하루 다건 매수(별지점/평단 + 아래 여러 단). 동시 다건 주문·매매거부(20%) 처리.
  • 소진모드(리버스) 별도 구현 필요(자료 확보 후).

선행 과제 상세 — 체결 동기화 자동화 (V4.0의 전제조건)

V4.0의 모든 상태 갱신은 실제 체결 이벤트로만 정의된다 — 매수 체결 +1/+0.5, 쿼터매도 ×0.75, 지정가매도 ×0.25 (§2). 체결을 자동·신뢰 가능하게 반영하지 못하면 T가 실제 포지션과 어긋나고, 어긋난 T는 별지점·매수금·매도수량 전부를 오염시킨다. 즉 체결 동기화는 V4.0의 부가 기능이 아니라 성립 조건이다. V2.2 운영에서도 누적 실현수익 표시와 사이클 종료(보유 0) 판정이 이 동기화에 의존한다.

실행 가능성 — $10,000이면 정수 주식으로 전 구간 작동한다. 평단 $50 앵커 기준 전반전 각 주문 2주(합 4주/일), 후반전 5주/일, 첫매수 4주. 각 주문 최소 1주 보장 조건은 ½회분($125) ≥ 최고 매수점(평단×1.1425), 즉 평단 ≤ 약 $109까지 유효(= 원금 ≥ 91.4×평단과 동일). 단 수량은 평단이 높을수록 줄어든다 — 예: 평단 $70.96(실전 TQQQ 사이클 수준)이면 전반 1주+1주, 후반 3~4주. 소수점 매수 없이도 성립하므로 토스 LOC 정수 주문으로 충분.
